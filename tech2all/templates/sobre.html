Sobre o Projeto

O Tech2All é um projeto acadêmico pensado para facilitar o acesso ao aprendizado de tecnologias voltadas ao desenvolvimento web.

Para isso, a plataforma oferece uma trilha de cursos introdutórios à engenharia de software. A sequência é pensada para poder ser consumida de forma fluida e complementar, visto que o conteúdo de cada curso se completa, construindo uma boa base de conhecimentos de programação. Por meio disso, o aluno torna-se capaz de dar seus primeiros passos como um desenvolvedor em formação.

Objetivos

Através de cursos progressivos de Python, JavaScript, HTML5 e CSS3, o aluno será capaz de criar projetos de sua preferência, utilizando HTML semântico, visual atrativo e um sistema de rotas funcional.

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias: Python, HTML5, CSS3 e o micro-framework Flask.
