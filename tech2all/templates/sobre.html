Tech4All

Seu curso de tecnologia

Sobre

O Tech4All é um projeto acadêmico pensado para facilitar o acesso ao aprendizado de tecnologias voltadas ao desenvolvimento web.

Para isso, a plataforma oferece uma trilha de cursos introdutórios à tecnologias inspiradas em aulas de barachelado do primeiro período de engenharia de software, pensados para passar conhecimento de forma fluida, visto que são conhecimentos que se completam, e organizada (Cada etapa do curso serve como pré requisição para a próxima, até que o aluno seja capaz de dar os seus primeiros passos.)

Objetivos

Através de cursos progressivos de python, javaScript, html e css3, o aluno deve ser capaz de criar uma página, de sua preferência, com html semântico, um visual atrativo e sistema de rotas funcional.

Tecologias utilizadas

Este projeto foi desenvolvido com tecnologia python, html5, css3 e flask.
