Sobre

O Tech4All é um projeto acadêmico pensado para facilitar o acesso ao aprendizado de tecnologias voltadas ao desenvolvimento web.

Para isso, a plataforma oferece uma trilha de cursos introdutórios a engenharia de software. A sequência é pensada para poder ser consumida de forma fluida e complementar (Visto que o conteúdo de cada curso se completa, construindo, assim, uma boa base de conhecimentos de programação). Por meio disso, o aluno deve tornar-se capaz de dar seus primeiros passos como um desenvolvedor em formação.

Objetivos

Através de cursos progressivos de python, javaScript, html e css3, o aluno deve ser capaz de criar projetos, de sua preferência, com html semântico, um visual atrativo e sistema de rotas funcional.

Tecologias utilizadas

Este projeto foi desenvolvido com tecnologia python, html5, css3 e flask.
